PHP syntax cheatsheet #1

literal values

number, string, boolean, float, array, object, null

123; 'abc'; true; false; 45.6; [1,2]; new Foo; null;

values that equal boolean false:

false, 0, null, '', []

values that equal boolean true:

true, 1, -1, 321, 'abc', [0]

operators

form expressions together with their operands

arithmetical operators

+ - * / % ( ) . ++ --

assignment operators

= += -= *= /= %= .=

comparison operators

== != < > <= >= ===

logical operators

&& || AND OR !

expressions

combination of one or more values, constants, variables, operators, and functions that are evaluated to produce another value

examples

1; 1+1; $foo; $foo+1; ($foo+1)*$bar; ($foo+1)*$bar+baz();

conditions

make decisions to decide what code (if any) should be run

if(expression) { // do this if expression is true } if(expression) { // do this if expression is true } else { // otherwise do this } if(expression1) { // do this if expression1 is true } elseif(expression2) { // do this if expression1 is false but expression2 is true } else { // otherwise do this }

loops

sequence of statements which is specified once but which may be carried out several times in succession

while, do ... while

while(expression) { // do this while expression is true } do { // do this while expression is true, but at least once } while(expression);

for

for(initialization; expression; change) { // do this while expression is true // at the beginning initialization is executed // after every iteration change is executed }

foreach (where $foo is an array)

foreach($foo as $bar) { // do this for each item in $array // $bar contains the value of the current item } foreach($foo as $baz => $bar) { // $baz contains the value of the current key // $bar contains the value of the current item }

variables

Hold values. Are used just like the values they hold.

create variable

$foo; $foo = 123;

use variable

$bar = 1 + $foo;

functions

sequence of program instructions that perform a specific task, packaged as a unit

function declaration

function foo() { // do this whenever the function is called } function bar($param1, $param2) { // variables $param1 and $param2 are available } function baz() { return $qux; // return a value } function xyzzy($param1 = 123) { // variable $param1 is available // if called without an argument, value of $param1 is 123 } function waldo(& $param1) { // variable $param1 is available // any change to it would also happen to the argument }

calling a function

foo(); bar(123, 'fred'); $foo = baz(); xyzzy(456); xyzzy(); waldo($argument1); // $argument1 may have been changed by this

arrays

complex data type describing a collection of elements

create

$foo = []; // empty array $bar = [123,'abc',true]; // with values $baz = ['xyz' => 123, 3 => 'abc']; // also specifying keys

add values

$foo[] = 'def'; // under next available numeric key $bar['xyz'] = 'fred'; // under a specific key

access values

echo $foo[0]; echo $bar['xyz'];

useful functions

array_push(); array_pop(); array_shift(); array_unshift(); sort(); ksort(); asort(); usort(); rsort(); krsort(); arsort(); array_merge(); array_search(); array_key_exists();
