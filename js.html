JavaScript syntax cheatsheet #1

literal values

number, string, boolean, float, array

123; 'abc'; true; false; 45.6; [1,2];

values that equal boolean false:

false, 0, null, '', []

values that equal boolean true:

true, 1, -1, 321, 'abc', [0]

operators

form expressions together with their operands

arithmetical operators

+ - * / % ( ) . ++ --

assignment operators

= += -= *= /= %= .=

comparison operators

== != < > <= >= ===

logical operators

&& || AND OR !

expressions

combination of one or more values, constants, variables, operators, and functions that are evaluated to produce another value

examples

1; 1+1; foo; foo+1; (foo+1)*bar; (foo+1)*bar+baz();

conditions

make decisions to decide what code (if any) should be run

if(expression) { // do this if expression is true } if(expression) { // do this if expression is true } else { // otherwise do this } if(expression1) { // do this if expression1 is true } else if(expression2) { // do this if expression1 is false but expression2 is true } else { // otherwise do this }

loops

sequence of statements which is specified once but which may be carried out several times in succession

while, do ... while

while(expression) { // do this while expression is true } do { // do this while expression is true, but at least once } while(expression);

for

for(initialization; expression; change) { // do this while expression is true // at the beginning initialization is executed // after every iteration change is executed }

variables

Hold values. Are used just like the values they hold.

create variable

var foo; foo = 123;

condensed form

var foo = 123;

use variable

bar = 1 + foo;

functions

sequence of program instructions that perform a specific task, packaged as a unit

function declaration

function foo() { // do this whenever the function is called } function bar(param1, param2) { // variables param1 and param2 are available } function baz() { return qux; // return a value }

calling a function

foo(); bar(123, 'fred'); bar(123); // param2 will be undefined var foo = baz();

Scope

Scope determines the accessibility (visibility) of variables.

global variables

var foo = 'asd'; // variable foo available here function bar(){ // variable foo available here } // variable foo available here

local variables

function bar(){ var foo = 'asd'; // variable foo available here } // variable foo NOT available here

shadowing global variables

Creating a local variable with the same name as global.

var foo = 'asd'; // variable foo available here with value 'asd' function bar(){ var foo = 'qwe'; // variable foo available here with value 'qwe' } bar(); // variable foo available here with value 'asd' var foo = 'asd'; // variable foo available here with value 'asd' function bar(){ foo = 'qwe'; // variable foo available here with value 'qwe' } bar(); // variable foo available here with value 'qwe'

arrays

complex data type describing a collection of elements

create

var foo = []; // empty array var bar = [123,'abc',true]; // with values

store values

bar[0] = 'fred'; // under a specific key

access values

foo[0];

useful functions

These functions are defined on Array.prototype object.

Array.prototype.push(); Array.prototype.pop(); Array.prototype.shift(); Array.prototype.unshift(); Array.prototype.sort(); Array.prototype.indexOf();

Example of using:

bar[0] = 'fred'; // under a specific key var value = bar.pop();
